# Mini Gestionnaire d’Annonces (CLI) – Python

## Objectif
Application en ligne de commande pour gérer des annonces (CRUD).

## Fonctionnalités
- Ajouter une annonce (dict dans list)
- Lister les annonces
- Rechercher par mot-clé (lower/strip/in)
- Supprimer par id (pop)
- Tags uniques (set)
- Catégories fixes (tuple)

## Exécution locale
```bash
python annonces_cli.py
